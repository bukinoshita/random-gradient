> Generate beautiful random gradients

<br/>
<img src="https://cldup.com/S4sruO8J60.png" width="600"/>

## Install
```
$ npm install --save random-gradient
```

## Usage
```js
import gradient from 'random-gradient'

gradient(uid)
// => linear-gradient(45deg, #d4f906 10%, #06d4f9 100%)
```

## API
### randomGradient(uid)

#### uid
Type: `string`<br/>
Required
